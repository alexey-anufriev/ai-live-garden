# GEMINI.md

Follow the project constitution in `AGENTS.md`.

For autonomous evolution runs, the Evolve workflow normally provides a compact context bundle generated by `scripts/build-evolve-context.sh`. Treat that bundle as the first-pass operating context because it is derived from:

* `AGENTS.md`
* `GEMINI.md`
* `README.md`
* `agent/state.md`
* `agent/requests.md`
* the latest active entries directly under `agent/journal/`, excluding `agent/journal/archive/`
* the latest active summaries under `agent/summaries/`, excluding all `archive/` folders
* `data/garden-state.txt`

This is a Java 25 / Maven project.

The compact bundle is the operational checklist for normal Evolve workflow runs. Inspect raw source files, raw state, or additional memory files only when needed for the chosen task.

Do not modify `AGENTS.md`, `GEMINI.md`, or `.github/`.
